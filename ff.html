Null-Cents

Features

  • Stores data in localStorage so expenses remain after refreshing.
  • Validates input to prevent empty names or invalid amounts.
  • Auto updates the total after each expense.

What Is This?

This is a simple Expense Tracker web application built using:
HTML (for the structure), CSS (for basic styling), JavaScript (for functionality), localStorage (for saving data in the browser).

It allows you to:

  • Enter an expense name and amount.
  • Add it to a list.
  • View all your expenses.
  • See the total cost.
  • Save all this information permanently in your browser (even if you close or refresh the page).

Benefits of This Expense Tracker

  • Easy to Use: Simple user interface for tracking daily expenses.
  • Data Persistence: Uses localStorage, so your data won't disappear when you reload or close the tab.
  • Private: Data is stored locally in your browser – not shared online.
  • Fast Setup: Doesn’t require any backend or server. Just open the file in a browser.
  • Customizable: Easily add features like delete, categories, filters, or even export to CSV.
  • Educational: Great small project to learn HTML, JavaScript, and browser storage.
